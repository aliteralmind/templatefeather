Template Featherweight: A simplistic Java template-rendering engine, providing for plain gaps only, such as {@code "Hello, %name%!"}--it has no advanced features such as if-logic, looping, or expressions--Templates may be output in full, after all gaps have been filled, or they may be auto-rendered, such that they are gradually output as each gap is filled.

"Featherweight" is a reaction to all the so-called "lightweight" Java templating solutions, all of which have advanced features.

  Installation for using Template Featherweight

This version is ${project_version_for_jd}.

  • Release notes
  • Current version at aliteralmind.com
  • Hosted by GitHub: Project page, issue tracker

    • Template Featherweight requires Java 1.7 (it may be compatible with earlier versions).

    (For building, see XBN-Java's installation instructions. The Template Featherweight build process is derived from XBN-Java's.)

    Apache Maven

    <dependency>
   <groupId>com.github.aliteralmind</groupId>
   <artifactId>templatefeather</artifactId>
   <version>^{project_version_for_jd}</version>
</dependency>

    Downloads

    1. The Template Featherweight jar, with no external dependencies (choose exactly one):
      1. {@code templatefeather_^{project_version_for_jd}-all.jar}: Core classes, example code, and unit tests: local copy, aliteralmind.com
      2. {@code templatefeather_^{project_version_for_jd}.jar}: Core classes only: local copy, aliteralmind.com
    2. {@code templatefeather_^{project_version_for_jd}_dependency_jars.jar}: All external jars required to use Template Featherweight: local copy, aliteralmind.com
    3. {@code templatefeather_^{project_version_for_jd}-sources.jar}: Source code, including all files necessary for building Template Featherweight: local copy, aliteralmind.com
    4. {@code templatefeather_^{project_version_for_jd}-javadoc.jar}: JavaDoc documentation for local installation: local copy, aliteralmind.com

    Adding the following elements to your classpath will install Template Featherweight:

    INSTALLATION_DIR\templatefeather-${project_version_for_jd}_dependency_jars\*
INSTALLATION_DIR\templatefeather-${project_version_for_jd}-all.jar

    (or templatefeather-${project_version_for_jd}.jar)
