Template Featherweight: A simplistic Java template-rendering engine, providing for plain gaps only, such as {@code "Hello, %name%!"}--it has no advanced features such as if-logic, looping, or expressions--Templates may be output in full, after all gaps have been filled, or they may be auto-rendered, such that they are gradually output as each gap is filled.

"Featherweight" is a reaction to all the so-called "lightweight" Java templating solutions, all of which have advanced features.

Installation instructions for using Template Featherweight

(For building, see XBN-Java's installation instructions. The Template Featherweight build process is derived from XBN-Java's.)

Adding the following elements to your classpath will install Template Featherweight:

  1. {@code INSTALLATION_DIR\templatefeather\z_build\jar_dependencies\commons-io-2.4.jar}
  2. {@code INSTALLATION_DIR\templatefeather\z_build\jar_dependencies\commons-lang3-3.3.2.jar}
  3. {@code INSTALLATION_DIR\templatefeather\z_build\jar_dependencies\xbnjava_0p1p0_templatefeather_needed_only.jar}
  4. {@code INSTALLATION_DIR\templatefeather\z_build\jar_dependencies\templatefeather_0p1p0_all.jar}

Details and alternatives:

  1. Template Feather: Exactly one of the following must be used (The above classpath-string assumes you've chosen {@code "templatefeather_0p1p0_all.jar"}):
    1. {@code templatefeather_0p1p0_all.jar}: Core classes, example code, and unit tests
    2. {@code templatefeather_0p1p0_core.jar}: Core classes only
  2. XBN-Java: Exactly one of the following must be used (The above classpath-string assumes {@code "xbnjava_0p1p0_templatefeather_needed_only.jar"}):
    1. {@code xbnjava_0p1p0_templatefeather_needed_only.jar}: Contains exactly those classes as required by Template Featherweight. If you have no need for other XBN-Java beyond Template Featherweight, choose this. It's about 1/3 of the size of {@code xbnjava_0p1p0_core.jar}.
    2. {@code xbnjava_0p1p0_core.jar}: All classes except example code and unit tests
    3. {@code xbnjava_0p1p0_all.jar}: All classes
    Warning: If you choose either {@code "core"} or {@code "all"}, this must also be added to your classpath:
        {@code INSTALLATION_DIR\templatefeather\z_build\jar_dependencies\commons-collections4-4.0.jar}
